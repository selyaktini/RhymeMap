# RhymeMapper History

## v0.1 (2025-03-14)
- First MVP release.
- Automatic label assignment (A, B, C...) to lines based on rhyme groups.

## v0.2 (2025-04-08)
- Replaced nucleus‑only rhyme detection with full syllable analysis (nucleus + coda).
- Used `syllabify` to extract syllables and their phonetic components.
- Introduced consonant families to handle slant rhymes.
- Color only terminal syllables whose rhyme signature appears at least 3 times.
- Removed line labels (`[A], [B] ..`) from the output display.
